# skill-creator

## Overview
When you need to create a new skill for OpenCode

## Process
1. Define skill name and description
2. Identify trigger conditions
3. Outline required input data
4. Document step-by-step procedures
5. Define expected output format

## Examples
- Creating custom domain skills
- Adding new tool capabilities
- Documenting team-specific workflows

## Limitations
Requires understanding of skill framework
