## I. Information Architecture School (01-04)
> Philosophy: "Data is not decoration, it is a building material"

### 01. Pentagram - Michael Bierut Style
**Philosophy**: Type is language, the grid is thought
**Key characteristics**:
- Extremely restrained color use (black and white + 1 brand color)
- A modern interpretation of the Swiss grid system
- Typography as the primary visual language
- Strategic use of negative space (60%+ whitespace)

**Prompt DNA**:
```
Pentagram/Michael Bierut style:
- Extreme typographic hierarchy, Helvetica/Univers family
- Swiss grid with precise mathematical spacing
- Black/white + one accent color (#HEX)
- Information architecture as visual structure
- 60%+ whitespace ratio
- Data visualization as primary decoration
```

**Representative work**: Hillary Clinton 2016 campaign identity
**Search keywords**: pentagram hillary logo system

## Prompt Usage Guide

**Combination formula**: `[Style Prompt DNA] + [Scene template (see scene-templates.md)] + [Specific content]`

### Core Principle: Describe Mood, Not Layout

The key to AI image generation: short prompts > long prompts. Three sentences describing mood and content work better than 30 lines of layout detail.

| Diversity-killing wording | Creativity-triggering wording |
|----------------|----------------|
| Specify color ratios (60% / 25% / 15%) | Describe mood ("warm like Sunday morning") |
| Prescribe layout positions ("title centered, image on the right") | Reference a specific aesthetic ("Pentagram editorial feel") |
| Restrict character poses and expressions | Let AI interpret the style naturally |
| List every visual element that must appear | Describe what the viewer should feel |

### Good / Bad Examples

**Bad - Over-constrained (AI output becomes empty and flat):**
```
Professional presentation slide. Dark background, light text.
Title centered at top. Two columns below. Left column: bullet points.
Right column: bar chart. Colors: navy 60%, white 30%, gold 10%.
Font size: title 36pt, body 18pt. Margins: 40px all sides.
```

**Good - Mood-driven (results are varied and textured):**
```
A data visualization that feels like a Bloomberg Businessweek
editorial spread. The key number "28.5%" should dominate the
composition like a headline. Warm cream tones with sharp black
typography. The data tells a story of dramatic channel shift.
```

### Choosing an Execution Path

Choose based on the "Best Path" column in the quick reference table:
- **AI Generation**: for styles with explicit visual elements (06/07/12/13/14/15/16/20), generate directly with Gemini/Midjourney
- **HTML Rendering**: for styles that depend on precise typography (01/03/04/10/11/17/18), use code to control data and layout
- **Hybrid**: use HTML for the structural layout + AI for illustrations/backgrounds (02/05/08/09/19)

### Quality Control

1. ❌ Do not write "in the style of Pentagram" directly -> ✅ describe specific design characteristics instead
2. Text often breaks in AI generation -> replace text afterward
3. Proportions are easily distorted -> specify the aspect ratio clearly
4. Generate 3-5 variations first, then refine the best one

**Default aesthetic no-go zones** (the user may override these for their own brand):
- ❌ Cyber neon / dark blue backgrounds (`#0D1117`)
- ❌ Personal signatures / watermarks on cover images
