# algorithmic-art

## Overview
When users request creating generative art, algorithmic art, flow fields, or particle systems

## Process
1. Create algorithmic philosophy document (4-6 paragraphs)
2. Design parameters for the generative system
3. Implement p5.js sketch with seeded randomness
4. Build interactive HTML artifact with parameter controls
5. Add seed navigation for exploration

## Examples
- Creating interactive generative art
- Building algorithmic visualizations
- Exploring parameter spaces with seeded art

## Limitations
Requires understanding of p5.js; computational complexity for real-time rendering
