# Brand Guidelines

Apply consistent brand colors, typography, and visual identity to all outputs.

## Brand Colors

| Token | Hex | Usage |
|-------|-----|-------|
| primary | #D946EF | Main brand color |
| secondary | #8B5CF6 | Accent |
| neutral | #1E293B | Text |
| background | #F8FAFC | Background |

## Typography

- Headings: Inter Bold
- Body: Inter Regular
- Code: JetBrains Mono

## Usage

Always apply brand tokens when creating:
- UI components
- Marketing materials
- Presentations
- Documents
