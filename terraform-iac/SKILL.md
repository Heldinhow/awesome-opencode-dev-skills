# terraform-iac

## Overview
When you need to create, modify, or manage cloud infrastructure programmatically

## Process
1. Initialize Terraform working directory with terraform init
2. Create or modify .tf configuration files with desired resources
3. Review execution plan with terraform plan
4. Apply changes with terraform apply
5. Verify infrastructure state with terraform show

## Examples
- Setting up AWS/GCP/Azure infrastructure
- Managing reproducible development environments
- Infrastructure versioning and auditing

## Limitations
Not suitable for single-server deployments; requires cloud provider access
