# copilot-delegate-subagent

## Overview
When you need to offload work to specialized subagents

## Process
1. Define the task or use custom agent
2. Delegate using direct, fleet, or sequential patterns
3. Monitor with /tasks command
4. Review aggregated results
5. Integrate into main workflow

## Examples
- Running parallel independent tasks
- Using specialized expert agents
- Breaking complex tasks into focused subtasks

## Limitations
Context isolation may limit cross-task awareness
