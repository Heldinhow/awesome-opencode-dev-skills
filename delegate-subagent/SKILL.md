# delegate-subagent

## Overview
When a task needs focused attention, specialized knowledge, or independent execution

## Process
1. Identify independent task suitable for delegation
2. Provide clear task description with context
3. Invoke subagent execution
4. Review returned results
5. Integrate changes into main workflow

## Examples
- Running focused refactoring in isolation
- Executing specialized tasks (tests, docs, migrations)
- Parallel independent work items

## Limitations
Not suitable for tasks requiring main context or real-time collaboration
