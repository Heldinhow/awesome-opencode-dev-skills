# dotnet-mediatr

## Overview
When implementing CQRS, need decoupled handlers, or pipeline behaviors

## Process
1. Install MediatR package
2. Define request/command records
3. Implement request handlers
4. Add pipeline behaviors (logging, validation)
5. Use in controllers for clean code

## Examples
- CQRS implementation
- Clean controller architecture
- Pipeline behaviors for cross-cutting concerns

## Limitations
Additional abstraction layer; may be overkill for simple apps
