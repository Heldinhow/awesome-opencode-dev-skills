# prisma-orm

## Overview
When working with databases in TypeScript/Node.js applications

## Process
1. Initialize Prisma project
2. Define schema in schema.prisma
3. Generate Prisma client
4. Use generated client for queries
5. Run migrations to apply schema

## Examples
- Type-safe database access
- Database migrations
- Rapid prototyping with ORMs

## Limitations
Additional abstraction layer; migration lock-in
