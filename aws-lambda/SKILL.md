# aws-lambda

## Overview
When you need to create event-driven serverless functions

## Process
1. Create Lambda function with runtime
2. Define handler function
3. Configure event triggers
4. Set up environment variables
5. Deploy and test with mock events

## Examples
- Event-driven processing
- API backends
- Data processing pipelines

## Limitations
Cold start latency; limited execution time; vendor lock-in
